# Tetris Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a basic playable Tetris section to the Flutter examples app, with model code demonstrating Factory, Bridge, Builder, Observer, and Iterator patterns.

**Architecture:** Keep game logic in pure Dart models under `lib/model/tetris/`. Keep Flutter widgets in `lib/tetris/`; widgets render state and forward commands to `TetrisGame`. `NextPieceIterator` owns fair random piece sequencing; `PileBuilder` creates immutable-ish pile transitions and emits events consumed by score/next logic.

**Tech Stack:** Flutter, Dart, `CustomPainter`, `flutter_test`, existing `flutter_lints`.

---

## Decisions Locked

- UI is a game only, not a visual pattern explainer.
- Rules: auto fall, move, rotate, soft drop, lock, line clear, score, next piece, game over, restart.
- Controls: buttons plus gestures.
- `NextPieceIterator`: generates fair bags of 140 entries: 20 of each piece type. Regenerates a new fair bag when exhausted.
- `PileBuilder`: builds the next `Pile`, clears lines, and returns/publishes events. It does not know `Score` or UI.
- `Pile` event observers: `pieceLocked`, `linesCleared(n)`.

enum TetrisPieceType { i, o, t, s, z, j, l }

class CellOffset {
  const CellOffset(this.x, this.y);
  final int x;
  final int y;
}

abstract class PieceBehavior {
  List<CellOffset> cellsForRotation(int rotation);
}

class TetrisPiece {
  const TetrisPiece({required this.type, required this.behavior, this.rotation = 0});

  final TetrisPieceType type;
  final PieceBehavior behavior;
  final int rotation;

  List<CellOffset> get cells => behavior.cellsForRotation(rotation);
  TetrisPiece rotate() => TetrisPiece(
        type: type,
        behavior: behavior,
        rotation: (rotation + 1) % 4,
      );
}
```

class NextPieceIterator {
  NextPieceIterator({required TetrisPieceFactory factory, Random? random})
      : _factory = factory,
        _random = random ?? Random() {
    _regenerateBag();
  }

  final TetrisPieceFactory _factory;
  final Random _random;
  final List<TetrisPieceType> _bag = [];
  int _index = 0;

  TetrisPiece next() {
    if (_index >= _bag.length) _regenerateBag();
    return _factory.create(_bag[_index++]);
  }
}
```
